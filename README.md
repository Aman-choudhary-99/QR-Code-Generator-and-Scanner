# QR Code Generator and Scanner

A simple web app to generate QR codes from text/URLs and scan QR codes from uploaded images.  
Built with HTML, CSS and vanilla JavaScript using **qrcodejs** and **jsQR**.

## Features

- Generate QR codes from any text or URL
- Download generated QR code as a **PNG** image
- Upload an image (PNG/JPEG) that contains a QR code and **decode** it
- Copy the decoded QR result to the clipboard
- Fully client-side â€“ no backend / server required
- Simple, responsive layout suitable for learning and portfolio projects
